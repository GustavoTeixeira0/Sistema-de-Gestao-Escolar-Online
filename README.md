# Sistema-de-Gest-o-Escolar-Online

Nome do projeto: Sistema de Gestão Escolar Online

1 – Equipe do projeto (nome completo) - máximo 5 pessoas.
Gustavo Teixeira Magalhães
2 – Breve resumo sobre o projeto.
Desenvolvimento de um sistema online para gerenciar informações escolares, como alunos, notas e frequência.
3 – Formulação do Problema
Dificuldade no controle e organização de dados escolares de forma eficiente.
4 – Justificativa
A informatização melhora a organização, acesso e segurança das informações.
5 – Objetivo Geral
Criar um sistema digital para facilitar a gestão escolar.
6 – Objetivos específicos (mínimo 5)
-	Desenvolver cadastro de alunos
-	Criar controle de notas
-	Implementar registro de frequência
-	Garantir acesso seguro ao sistema
-	Gerar relatórios automáticos

7 – Produto esperado
Um sistema funcional acessível via computador ou celular para gestão escolar.
